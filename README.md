# Multimodal Emotion Classifier

A deep learning-based emotion classifier designed to detect emotions from facial expressions using convolutional neural networks (CNN), and using KAN (Kolmogorovâ€“Arnold Network) method to train and create the model architecture
---

## **Features**

- Emotion classification for 7 classes: Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise.
- Data augmentation for robust training.
- Supports grayscale image input for resource efficiency.
- Flexible model architecture with options to add advanced regularization.

---

## **Dataset**

- **FER2013**:
  - Source: Kaggle.
  - Total Images: 35,887.
  - Classes: Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise.
  - Resolution: 48x48 (grayscale).
